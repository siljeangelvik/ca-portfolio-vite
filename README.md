## Getting started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:siljeangelvik/ca-portfolio-vite.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run and preview the app:

```bash
npm run dev
```
